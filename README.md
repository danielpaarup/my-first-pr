# my-first-pr

A practice repository for learning the pull request workflow on GitHub.

## What this is

This repo exists to get comfortable with the full PR cycle: create a branch, make a change, open a pull request, and merge it.

## Getting started

1. Clone the repo: `git clone https://github.com/danielpaarup/my-first-pr.git`
2. Create a branch: `git checkout -b your-branch-name`
3. Make a change and commit it: `git commit -am "describe your change"`
4. Push: `git push origin your-branch-name`
5. Open a pull request on GitHub
